## Tuần 3 (04/05 – 10/05)

### Mục tiêu
- Tiếp tục tìm hiểu kiến thức về mạng trên nền tảng AWS.
- Nghiên cứu kiến trúc Amazon Virtual Private Cloud (VPC).
- Thực hành xây dựng hạ tầng mạng cơ bản trên AWS.

### Nội dung thực hiện

**04/05**
- Học Module 02-01 - AWS Virtual Private Cloud.
- Tìm hiểu khái niệm Amazon VPC và vai trò trong việc xây dựng hạ tầng mạng trên AWS.

**05/05**
- Học Module 02-02 - VPC Security and Multi-VPC features.
- Tìm hiểu các tính năng bảo mật trong VPC và mô hình Multi-VPC.

**06/05**
- Thực hiện Module 02-Lab03-03.1 - Create VPC.
- Tạo Virtual Private Cloud trên AWS.
- Tìm hiểu cách cấu hình CIDR Block cho VPC.

**07/05**
- Thực hiện Module 02-Lab03-01.1 - Subnets.
- Tìm hiểu Public Subnet và Private Subnet.
- Thực hiện Module 02-Lab03-03.2 - Create Subnet.
- Tạo Subnet trong VPC.

**08/05**
- Thực hiện Module 02-Lab10-02.1 - Generate Key Pair.
- Tạo Key Pair phục vụ kết nối Amazon EC2.
- Tìm hiểu vai trò của Key Pair trong việc xác thực truy cập máy chủ.

**09/05**
- Thực hiện Module 02-Lab03-01.3: Internet Gateway (IGW).
- Thực hiện Module 02-Lab03-03.3: Create an Internet Gateway.
- Tìm hiểu vai trò của Internet Gateway trong việc kết nối VPC với Internet.

**10/05**
- Ôn tập các nội dung đã thực hành trong tuần.
- Tổng hợp kiến thức về VPC, Subnet, Internet Gateway và Key Pair.

### Kết quả đạt được
- Hoàn thành Module 02-01 và Module 02-02.
- Hiểu kiến trúc Amazon VPC và các thành phần mạng cơ bản.
- Tạo thành công VPC, Subnet và Internet Gateway.
- Hiểu vai trò của Key Pair trong kết nối Amazon EC2.
