## Tuần 7 (01/06 – 07/06)

### Mục tiêu
- Nghiên cứu các dịch vụ AWS phù hợp cho dự án Budget Tracker.
- Phân tích yêu cầu nghiệp vụ quản lý chi tiêu cá nhân.
- Xây dựng phương án triển khai hệ thống trên AWS Serverless.

### Nội dung thực hiện

**01/06**
- Tìm hiểu kiến trúc Serverless trên AWS.
- Ôn tập Amazon API Gateway, AWS Lambda và Amazon DynamoDB.

**02/06**
- Phân tích yêu cầu của hệ thống Budget Tracker.
- Xác định các chức năng: Quản lý giao dịch, Quản lý ngân sách, Quản lý danh mục, AI phân tích chi tiêu, Cảnh báo vượt ngân sách.

**03/06**
- Nghiên cứu các dịch vụ AWS sẽ sử dụng: Amazon Route53, Amazon CloudFront, AWS WAF, Amazon S3, Amazon API Gateway, Amazon Cognito, AWS Lambda, Amazon DynamoDB, Amazon Bedrock, Amazon SNS, Amazon CloudWatch, Amazon SQS.

**04/06**
- Xây dựng kiến trúc triển khai theo mô hình Serverless.
- Đánh giá khả năng mở rộng và tính sẵn sàng.

**05/06**
- Thiết kế luồng xử lý người dùng.
- Thiết kế luồng AI phân tích chi tiêu.

**06/06**
- Phân chia công việc cho các thành viên.
- Lập kế hoạch triển khai Backend, Frontend và AWS.

**07/06**
- Tổng hợp tài liệu nghiên cứu.
- Chuẩn bị cho giai đoạn thiết kế kiến trúc.

### Kết quả đạt được
- Xây dựng phương án triển khai hệ thống Budget Tracker trên AWS.
- Xác định đầy đủ các dịch vụ AWS sử dụng.
- Hoàn thiện kế hoạch triển khai.
